OpReturn Algorithm

OpReturn: result = stack.pop() // grab return value (top of stack) frame = call_stack.pop() // remove current frame if call_stack.is_empty() → return Ok // script returned → done stack.truncate(frame.stack_base) // wipe everything from this call stack.push(result) // land result where callee slot was

Stack Slot Layout (any function)

base + 0 = <fun itself> // reserved — FunctionCompiler.locals[0] base + 1 = arg[0] // first argument base + 2 = arg[1] base + N = first local var // var declared inside body OpGetLocal slot → stack[base + slot] OpSetLocal slot → stack[base + slot] = peek_top()
